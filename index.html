POCHICO Q&A

よくある質問

Q1: 注文したらどのくらいできますか？

+

在庫があれば5日以内にはお渡しができると思います。
在庫がなければ2週間以上お時間をいただきます。

Q2: マジックのキャップだけ購入できますか？

+

キャップだけの注文はできません。

Q3: 名札1枚でも購入できますか？

+

はい。購入可能です。

Q4: 体操服を購入すると名札はついてきますか？

+

付いてきます。

Q5: 上靴の試し履きはできますか？

+

園にある上靴の見本のサイズで試し履きをしてもらっています。

Q6: 体操服の名札はどこにつけたらいいですか？ズボンにも名札をつけますか？

+

左の胸あたりにつけてください。
ズボンには名札は必要ありません。

Q7: 保護者証は中身のみ購入できますか？

+

ケースと中身はセット販売です。

Q8: 出席ブックのシールのみ購入できますか？

+

購入可能です。

Q9: 鍵盤ハーモニカの唄口は売っていますか？

+

はい。あります。

Q10: 自由画帳は100均で購入してもいいですか？

+

基本的には園のものを購入していただいております。

Q11: 上靴が（園指定）大きかったのでサイズ変更できますか？

+

氏名などの記載をしていなければ可能です。
大きい場合、中敷を入れてみてはいかがですか？
とも伝えています。

Q13: 体操服の140以上はありますか？

+

あります。ただし在庫はなく注文になるため
届くまでに時間がかかる場合があります。

Q14: 返品交換はできますか？

+

記名の前であれば可能です。

Q15: 上靴は園の指定のものですか？

+

園のものでも市販のものでも構いませんが、靴の底が黒いものは避けてください。

Q16: 家にあるものを使っていいですか？

+

代用できるものがあれば大丈夫ですが、一度園にご相談ください。。

Q17: カラーペン又は色鉛筆を１本で購入可能ですか？

+

購入可能です。

Q18: 名前は漢字で書かないといけないですか？

+

漢字表記が基本ですが、すでに記入されているものが平仮名等でもそれで大丈夫です。
